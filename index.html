Hướng Dẫn Cài Đặt Android

Làm theo các bước sau để cài đặt Jinro trên thiết bị Android của bạn.

1. Cài đặt Termux

Tải và cài đặt Termux từ Github (tránh bản Play Store). Chọn universal hỗ trợ tất cả cấu trúc.

2. Cài đặt phần mềm đi kèm

3. Chạy Lệnh Cài Đặt Menu

Mở Termux, sao chép và dán vào các lệnh sau:

termux-setup-storage
apt update && apt upgrade -y
pkg i -y python python-pip openjdk-17 git
pip install mysql-connector-python gdown gspread google-auth
git clone https://github.com/ki3tngu/jinro

4. Tải Dữ Liệu & Chạy Menu

Sau khi cài đặt xong, di chuyển vào thư mục và chạy menu:

cd jinro
python main.py

Trong menu hiện ra, chọn tùy chọn số 5 để tải dữ liệu game cần thiết.

5. Sửa lỗi

CANNOT LINK EXECUTABLE "sh": "/data/data/com.termux/files/usr/lib/libtermux-exec-ld-preload.so" is 32-bit instead of 64-bit

termux-setup-storage
unset LD_PRELOAD
apt reinstall /sdcard/termux-exec_2_1.0-1_aarch64.deb -y
